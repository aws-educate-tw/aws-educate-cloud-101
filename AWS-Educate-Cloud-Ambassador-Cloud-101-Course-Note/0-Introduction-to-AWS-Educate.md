# 📍 What is AWS?

---

Amazon Web Services (AWS) 為全球最全面和廣泛採納的雲端平台，提供超過 200 種功能完整的服務，包含運算、儲存和資料庫等基礎設施技術，數百萬個客戶 —包括成長最快的新創公司、最大型企業(Facebook, Netflix)以及領先的政府機構—都使用 AWS 來降低成本、變得更靈活，且更迅速地創新。
